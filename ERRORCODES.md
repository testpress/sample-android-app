# Error Codes

### Error Code: 404

*   Invalid VideoID

### Error Code: 401

*   Invalid Access token

### Error Code: 6008

*   DRM license expired

### Error Code: 6004

*   Not able to get DRM license

### Error Code: 4003

*   Device does not support the video format

### Error Code: 3001

*   License not download for encrypted video
